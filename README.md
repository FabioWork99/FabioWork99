## 🚀 Projetos em Destaque

### 🚗 [Turbo Garagem - Calculador de Valor de Carros](https://github.com/FabioWork99/Calculador_de_valor_de_carros)
Uma API de Inteligência Artificial para previsão de preços de carros usados baseada em histórico de mercado. O sistema analisa características do veículo e estima um preço justo de forma instantânea.
